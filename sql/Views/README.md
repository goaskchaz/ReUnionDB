# Views

This folder contains SQL Server views used by ReUnionDB.

Views simplify reporting by combining data from one or more tables into reusable virtual tables.

## Naming Convention

Views begin with the prefix:

vw

Examples:

- vwClassmatesDied.sql
- vwCurrentAddresses.sql
- vwAttendanceSummary.sql

## Purpose

Views provide:

- Reporting
- Read-only data access
- Simplified queries
- Data abstraction
