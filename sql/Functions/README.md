# Functions

This folder contains SQL Server user-defined functions.

Functions return either a single value or a table and are intended to simplify reusable calculations.

## Naming Convention

Functions begin with:

ufn

Examples

- ufnCalculateAge.sql
- ufnFullName.sql
- ufnYearsSinceGraduation.sql

## Purpose

Functions provide reusable calculations that can be referenced throughout SQL queries and stored procedures.
