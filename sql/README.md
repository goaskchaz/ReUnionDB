# SQL Objects

This folder contains all SQL Server database objects used by the ReUnionDB database.

The SQL scripts are organized by object type to simplify development, deployment, version control, and maintenance.

## Folder Structure

| Folder | Purpose |
|---------|---------|
| Tables | Database table definitions. |
| Views | SQL views used for reporting and simplified data access. |
| Stored Procedures | Business logic, reporting, and data manipulation procedures. |
| Functions | User-defined scalar and table-valued functions. |
| Triggers | Database triggers that execute automatically after data changes. |
| Scripts | Utility scripts used for maintenance, migrations, testing, and administration. |
| SeedData | Sample and initial data used to populate lookup and reference tables. |
