# Stored Procedures

This folder contains stored procedures used by ReUnionDB.

Stored procedures encapsulate business logic, reporting, data imports, maintenance, and administrative tasks.

## Naming Convention

Stored procedures begin with:

usp

Examples

- uspInsertClassmate.sql
- uspUpdateAddress.sql
- uspReunionStatistics.sql

## Purpose

Stored procedures provide:

- Data validation
- Consistent updates
- Reporting
- Batch processing
- Performance optimization
