# Scripts

This folder contains miscellaneous SQL scripts used during database administration and development.

Scripts are generally executed manually and are not permanent database objects.

Examples include:

- Data cleanup
- Maintenance
- Security
- Performance tuning
- Database migrations
- Reporting
- Testing

## Typical Files

- Backup.sql
- RebuildIndexes.sql
- FindDuplicates.sql
- UpdateStatistics.sql
